
API: OpenRouter

- Как получить apiKey:
зайти на сайт [OpenRouter](https://openrouter.ai/) => Gep api key => + New Key 

Перед запуском создайте в папке Back файл с названием ".env"
и вставте туда ваш токен OPENROUTER_API_KEY = ваш ключь 

# Запуск:

- В первом терминале
```
cd .\Back
```

```
npm install
```

```
node server.js
```

- Во втором терминале
```
cd .\Front
```

```
npm install
```
```
npm run dev
```
